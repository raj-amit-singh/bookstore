# Bookstore

Modern web app clean architecture with NestJS + MongoDB + Angular

## Technologies
- Backend
  - NodeJs/NestJs TypeScript
  - MongoDB/Mongoose
  - Jest (testing)
- Front-end
  - Angular 16
  - CSS
  - Cypress (testing)
